# Data

Training files are included in this folder (DengAI competition format).

| File | Description |
|------|-------------|
| `dengue_features_train.csv` | Weather, NDVI, station features (weekly) |
| `dengue_labels_train.csv` | Target `total_cases` |
| `submission_format.csv` | Submission layout (competition) |

Source: [DengAI – Predicting Disease Spread](https://www.drivendata.org/competitions/44/dengai-predicting-disease-spread/) (DrivenData / U.S. CDC).

Place `dengue_features_test.csv` here if you want to run inference on the test set (not required for the course notebook).
